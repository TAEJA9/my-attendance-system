✨ 실시간 출석 체크 시스템 ✨

강의, 모임, 행사 등 어디서든 간편하게 출석을 확인하고 관리할 수 있는 스마트한 웹 시스템입니다. 복잡한 명단 관리 없이 실시간으로 출석 현황을 파악할 수 있습니다.

주요 기능

간편한 출석

이름만 입력하면 출석 완료! 복잡한 절차 없이 누구나 쉽게 사용할 수 있습니다.

실시간 현황

다른 기기에서도 출석 현황이 즉시 업데이트되어 모두가 최신 정보를 공유합니다.

출석 기록 관리

비밀번호로 보호되는 관리자 모드에서 출석 기록을 초기화할 수 있습니다.

기술 스택

HTML5 CSS3 (Tailwind CSS) JavaScript Google Firebase (Firestore) Google Firebase (Authentication) GitHub Pages
